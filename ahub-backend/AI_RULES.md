# Project Rules

Project: AHUB Backend

Stack:
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- JWT

Architecture:
- Router Layer
- Service Layer
- Model Layer
- Schema Layer

Rules:

1. Follow existing structure.
2. Never create duplicate files.
3. Explain modifications first.
4. Use type hints.
5. Use dependency injection.
6. Keep business logic in services.
7. Routers should be thin.
8. Use Pydantic V2.
9. Generate production-ready code.
10. Maintain backward compatibility.
